# Kernel Learning 🐧

This repository documents my journey studying the Linux kernel.

## Topics

- Kernel architecture
- Kernel modules
- Device drivers
- Process scheduling
- Memory management
- System calls

## Goal

Understand how the Linux kernel works internally and how operating systems interact with hardware.
